# プライバシーポリシー

最終更新日: 2026年3月19日

「アフィリエイトID自動付与」（以下「本拡張機能」）のプライバシーポリシーについて説明します。

## データの収集

本拡張機能は、ユーザーの個人情報や閲覧データを一切収集しません。

## データの保存

本拡張機能が保存するデータは以下のみです:

- **アフィリエイトID**: ユーザーが設定画面で入力したアソシエイトID
- **有効/無効設定**: 機能のON/OFF状態

これらのデータは `chrome.storage.sync` API を通じてChromeブラウザのローカル同期ストレージにのみ保存されます。Googleアカウントの同期が有効な場合、同じアカウントの他のChromeブラウザと設定が同期されます。

## 外部通信

本拡張機能は外部サーバーへの通信を一切行いません。すべての処理はブラウザ内で完結します。

## URL書き換え

本拡張機能は `chrome.declarativeNetRequest` API を使用して、Amazon.co.jp へのリクエスト時にURLの `tag` クエリパラメータを書き換えます。この処理はブラウザのネットワークレイヤーで行われ、閲覧履歴の記録や外部への送信は行いません。

## 権限の使用目的

| 権限 | 使用目的 |
|------|----------|
| `declarativeNetRequest` | URLへのアフィリエイトIDパラメータの付与 |
| `storage` | ユーザー設定の保存・同期 |
| `activeTab` | ポップアップでの現在タブ状態表示 |
| `host_permissions: *://*.amazon.co.jp/*` | Amazon.co.jpのURLのみを書き換え対象とするため |

## 第三者への提供

本拡張機能はユーザーのデータを第三者に提供、販売、共有することはありません。
