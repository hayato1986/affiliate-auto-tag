アフィリエイトID自動付与

プライバシーポリシー

最終更新日: 2026年3月19日

本拡張機能は、ユーザーの個人情報や閲覧データを一切収集しません。外部サーバーへの通信も一切行いません。

本拡張機能について

「アフィリエイトID自動付与」（以下「本拡張機能」）は、Amazon.co.jp を閲覧する際に、ユーザーが設定したアソシエイトIDをURLの tag パラメータに自動付与するChrome拡張機能です。

データの収集

本拡張機能は、以下を含むいかなるユーザーデータも収集・送信しません:

  • 閲覧履歴・アクセスしたURL
  • 個人を特定できる情報
  • Cookie・トラッキング情報
  • フォーム入力内容

データの保存

本拡張機能が保存するデータは以下の2種類のみです:

データ 内容 保存先
アソシエイトID ユーザーが設定画面で入力したID chrome.storage.sync
有効/無効設定 機能のON/OFF状態 chrome.storage.sync

これらのデータはChromeブラウザのローカル同期ストレージにのみ保存されます。Googleアカウントの同期が有効な場合、同じアカウントでログインしている他のChromeブラウザと設定が自動同期されます。

外部通信

本拡張機能は外部サーバーへの通信を一切行いません。アナリティクス、テレメトリ、クラッシュレポート等の送信機能は含まれていません。すべての処理はブラウザ内で完結します。

URL書き換えの仕組み

本拡張機能は Chrome の declarativeNetRequest API を使用して、Amazon.co.jp へのリクエスト時にURLのクエリパラメータを書き換えます。この処理はブラウザのネットワークレイヤーで宣言的に行われ、JavaScriptからページ内容にアクセスすることはありません。

権限の使用目的

権限 使用目的
declarativeNetRequest Amazon.co.jp へのリクエストURLにアフィリエイトIDパラメータを付与するため
storage ユーザーが設定したアソシエイトIDとON/OFF状態を保存・同期するため
activeTab ポップアップUIで現在のタブがAmazon.co.jpかどうかを表示するため
host_permissions:
*://*.amazon.co.jp/*		 Amazon.co.jp のURLのみを書き換え対象とするため（他サイトには影響しません）

第三者への提供

本拡張機能はユーザーのデータを第三者に提供、販売、共有、転送することはありません。

お子様のプライバシー

本拡張機能は13歳未満のお子様を対象としたものではありません。13歳未満のお子様から意図的に個人情報を収集することはありません。

ポリシーの変更

本プライバシーポリシーは、機能の追加や法令の変更に伴い更新される場合があります。重要な変更がある場合は、拡張機能のアップデートに合わせてお知らせします。
